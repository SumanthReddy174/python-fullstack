37. Implement a decorator that counts how many times a function is called.

What is this decorator for?

It helps us know how many times a function has been executed.
Each time we call the function, the counter increases.

def count_calls(func):
    count = 0  # Initialize counter

    def wrapper(*args, **kwargs):
        nonlocal count  # allows us to modify 'count' from outer function
        count += 1
        print(f"Function '{func.__name__}' called {count} times")
        return func(*args, **kwargs)  # Call the original function
    return wrapper
