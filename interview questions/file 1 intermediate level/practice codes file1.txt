3. Count vowels in a string.
program 

text = "Beautiful Day"
vowels = "aeiouAEIOU"
vowel_count = {}

for char in text:
    if char in vowels:
        vowel_count[char] = vowel_count.get(char, 0) + 1

print("Vowel counts:", vowel_count)



4.Write a program to check if a number is **prime**.
program:
num = 29

if num > 1:
    for i in range(2, num):
        if num % i == 0:   # divisible by some number
            print(num, "is NOT a prime number")
            break
    else:
        print(num, "is a PRIME number")
else:
    print(num, "is NOT a prime number")
